# Tanks

Bài tập lớn 5: Lập trình game - HCMUT

Mô tả
- Đây là bài tập lớn môn Lập trình game - HCMUT. Game này hiện thực các cơ chế: bắn xe tăng, multiplayer, AI cho bot.

Yêu cầu

Hướng dẫn mở project
1. Clone repository:
   git clone https://github.com/<your-username>/Tanks.git
2. Mở Unity Hub và chọn folder project `Tanks`.
